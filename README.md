# Uncanny Unova

Unova Red Redux: Uncanny Unova 

A remastered Unova Red with Fairy type, improved Learnsets, and rehauled SPECIAL stats for certain Pokemon. 

Pokemon Frolic and Pokemon Steadfast are spin offs of this project with Emolga (Pokemon Frolic) as the starter Pokemon and Lillipup (Pokemon Steadfast) as the starter Pokemon.
